Transition and Transversion Ratio Table

The table above, displayed in the Results section, presents the transition/transversion (Ti/Tv) ratios for each sequence pair. For example, when Query 1 is compared with Query 2, the Ti/Tv ratio is 1.378.

Nucleotide Matrix

The Nucleotide Matrix provides detailed insights into individual nucleotide substitutions between compared sequences. In the above example of query 1 and query 2 :

  • Nucleotide "A" was substituted with "C" six times.
  • Nucleotide "A" was substituted with "G" twenty four times.
  • Nucleotide "A" was substituted with "T" twice.

This matrix helps analyze mutation frequencies and different types of nucleotide changes.

Header and Query Indexing

At the bottom of the Results page, Variantis provides query indexing details, linking each query to the original FASTA header provided by the user.

Note: The order of queries in the results matches the order of sequences in the uploaded FASTA file.

Color-Coded Alignment Output

The color-coded alignment visually represents the nucleotide differences between two sequences.

Key highlights:

  • Identical nucleotides
  • Transitions (A ↔ G, C ↔ T)
  • Transversions (Purine ↔ Pyrimidine changes)
  • Gaps

Below the alignment, a statistics table and a pie chart provide an in-depth breakdown of these categories.

Summary Dashbord

The summary dashbord gives overall summary of the sequences including :

  • Nucleotide substitutions along with their percentages
  • Length of individual sequence pairs along with their length after alignment.

Run Details and Parameter Summary

The Run Details section provides:

  • The alignment program used (e.g., EMBOSS Needle or EMBOSS Stretcher).
  • The gap opening and extension penalties.
  • The exact command executed for the alignment.

Additionally, downloadable links are provided for:

  • Nucleotide Matrix
  • Transition/Transversion Matrix
  • Alignment Files for all queries
