Welcome to the Variantis Tutorial

Step 1: Upload Your FASTA File

To begin your analysis, upload a FASTA-formatted file containing your sequences.

Note: Only files in FASTA format are accepted.

Step 2: Understanding Alignment Parameters

Click on the "?" icon to view the default details about the parameters Variantis uses for pairwise sequence alignment.

Step 3: Customizing Parameters

Variantis allows customization of alignment settings:

  • Pairwise Alignment Program: Choose between EMBOSS Needle and EMBOSS Stretcher.
  • Gap Open & Gap Extension: Adjust values for optimal alignment.

Click on the "Custom Parameters" button, enter your values.

Step 4: Submitting Your File

Once your file is uploaded and parameters are set, click Submit to initiate the analysis.

Step 5: Viewing Results

After processing, you will be redirected to the Results page, which includes:

  • A table displaying transition/transversion ratios for each sequence pair.
  • A nucleotide matrix detailing individual nucleotide substitutions.
  • Header and query indexing information.

Step 6: Exploring Individual Outputs

For a detailed visualization of pairwise alignments, click on Individual Output:

  • Select the required query and subject sequences from the dropdown menu.
  • View the color-coded alignment, highlighting:
    • Identical nucleotides
    • Transitions (A ↔ G, C ↔ T)
    • Transversions (Purine ↔ Pyrimidine changes)
    • Gaps

Step 7: Summary Dashboard

This tab provides the overall substitution count as well as individual length of each pair for comparision.

Step 8: Accessing Run Details & Downloads

The Run Details section provides:

  • The alignment parameters used for analysis.
  • The exact command executed for the alignment.
  • Downloadable links for:
    • Nucleotide matrix
    • Transition/Transversion matrix
    • Alignment files for all queries
    • Summary Matrices
